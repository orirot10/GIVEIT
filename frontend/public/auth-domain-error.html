Fix Firebase Authentication Domain Error

Error message: Firebase: Error (auth/unauthorized-domain)

This error occurs because your current domain is not authorized in Firebase Authentication settings.

How to Fix

Option 1: Add Your Domain to Firebase

  1. Go to the Firebase Console
  2. Select your project (giveit-3003)
  3. In the left sidebar, click on Authentication
  4. Click on the Settings tab
  5. Scroll down to the Authorized domains section
  6. Click Add domain
  7. Add your domain (e.g., localhost, giveit-frontend.onrender.com)
  8. Click Add

Option 2: Use Email/Password Authentication

Until the domain is authorized, you can use email/password authentication instead of Google sign-in.

For Developers

If you're developing locally:

  1. Make sure localhost is added to the authorized domains
  2. If using Firebase Emulator, make sure it's properly configured
  3. Check that your Firebase configuration matches the project where you've authorized domains

After adding your domain, it may take a few minutes for the changes to take effect. Try refreshing the page after a few minutes.
